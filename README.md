# ­ЪјЊ рЃЋрЃўрЃЊрЃћрЃЮ рЃЎрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљ

рЃћрЃА рЃљрЃарЃўрЃА рЃЋрЃћрЃЉ-рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљ, рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЋрЃўрЃЊрЃћрЃЮ рЃЎрЃБрЃарЃАрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ, рЃЌрЃћрЃЏрЃћрЃЉрЃўрЃА рЃюрЃљрЃ«рЃЋрЃљ рЃЊрЃљ рЃЋрЃўрЃЊрЃћрЃЮрЃћрЃЉрЃўрЃА рЃДрЃБрЃарЃћрЃЉрЃљ.  
рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃА рЃљрЃЦрЃЋрЃА **рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃў**, рЃарЃЮрЃЏрЃџрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌрЃљрЃф рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЎрЃБрЃарЃАрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЌрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ.

## ­Ъџђ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### ­ЪЉеРђЇ­ЪјЊ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃ«рЃљрЃарЃћ

- рЃЌрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃЏрЃЮрЃюрЃљрЃЌрЃЋрЃљрЃџрЃў (­ЪЊ▒ рЃЏрЃЮрЃЉрЃўрЃџрЃБрЃарЃќрЃћ рЃЕрЃљрЃЏрЃЮрЃерЃџрЃўрЃЌ рЃЏрЃћрЃюрЃўрЃБ)
- рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃЌрЃћрЃЏрЃўрЃА рЃЋрЃўрЃЊрЃћрЃЮрЃА рЃЕрЃљрЃерЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃърЃџрЃћрЃћрЃарЃў
- рЃЌрЃћрЃЏрЃўрЃА рЃљрЃдрЃгрЃћрЃарЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ

### ­ЪћЉ рЃљрЃЊрЃЏрЃўрЃю рЃърЃљрЃюрЃћрЃџрЃў
- **рЃљрЃ«рЃљрЃџрЃў рЃЎрЃБрЃарЃАрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ** (рЃАрЃљрЃЌрЃљрЃБрЃарЃў, рЃАрЃБрЃарЃљрЃЌрЃў, рЃЌрЃћрЃЏрЃћрЃЉрЃў)
- **рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃЎрЃБрЃарЃАрЃўрЃА рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ**
- **рЃЎрЃБрЃарЃАрЃўрЃА рЃгрЃљрЃерЃџрЃљ**
- **рЃЌрЃћрЃЏрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ/рЃгрЃљрЃерЃџрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ рЃЎрЃБрЃарЃАрЃерЃў**

# ­ЪјЊ Video Courses Platform

This is a web application for browsing and watching video courses.  
The app includes an **Admin Panel** where courses and topics can be created, edited, and deleted.  
The admin panel is protected by a password.

---

## ­Ъџђ Features

### ­ЪЉеРђЇ­ЪјЊ User Side
- Browse list of available courses
- View details of a specific course
- See list of topics (­ЪЊ▒ collapsible on mobile)
- Watch embedded videos for each topic
- Read topic descriptions

### ­ЪћЉ Admin Panel
- **Create new course** (title, image, topics)
- **Edit existing courses**
- **Delete courses**
- **Add or remove topics** for a course
- Upload course images
- Protected by **password** (`123`)

---

## ­ЪЏа№ИЈ Tech Stack
- **Frontend:** React + TailwindCSS  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **Deployment:** Render
